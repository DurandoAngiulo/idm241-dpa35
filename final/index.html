Final: Description

Epic Games Store Url

Triggers

  • Hover over ANY Game Image
  • Hover over any category Button
  • Hover over ANY info button
  • Click on ANY info button
  • Click on ANY "X" button

Rules

INTERACTION 1. The Cursor Hovers In over any Game Image button:

  • Cursor changes to pointer
  • Hovered image expands by 30%
  • Game info textbox fades in towards bottom of hovered image

INTERACTION 2. The Cursor Hovers In over any game category:

  • Hovered category expands by 10%
  • category opacity increases by 25% and color changes to white

The Cursor Hovers Out of any game category:

  • Hovered category contracts by 10%
  • Category opacity decreases by 25% and color changes to grey

INTERACTION 3. The Cursor Hovers In over any info button:

  • Hovered button expands by 20%
  • Button opacity increases by 20% and color changes to white

INTERACTION 4. The Pointer clicks on any info button:

  • The Game Info Modal slides in from the bottom to the center of the screen, going from 0 to 95% opacity
  • pointer goes back to cursor

INTERACTION 5. The Cursor hovers on any "X" button:

  • The cursor goes back to pointer

INTERACTION 6. The Pointer Clicks on any "X" button:

  • The Game Info Modal slides down from the center to the bottom of the screen, going from 95 to 0% opacity
  • Game image shrinks back to original unhovered state
  • Game textbox shrinks and fades out
  • The pointer returns to a cursor
